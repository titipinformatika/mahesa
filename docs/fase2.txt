# Rencana Eksekusi Fase 2: Otentikasi Lanjutan & Data Inti

Dokumen ini berisi rincian pekerjaan untuk **Fase 2** proyek MAHESA. Pekerjaan telah dipecah menjadi beberapa *Issue* spesifik agar mudah dikerjakan oleh Programmer Junior atau Model AI.

Beberapa poin dari Fase 2 (seperti Skema Database dan Endpoint Login Dasar) telah diselesaikan pada Fase 1. Berikut adalah sisa pekerjaan yang harus diselesaikan.

---

### Issue #5: `[Backend] Sisa Endpoint Otentikasi & Integrasi Token FCM`
**Description:**
```markdown
**Tujuan:**
Melengkapi fitur otentikasi (refresh token, ganti sandi, reset) dan menyiapkan endpoint penyimpanan token Firebase Cloud Messaging (FCM) untuk push notification.

**Task List:**
- [ ] Buat Endpoint `POST /v1/otentikasi/keluar` (Logout).
- [ ] Buat Endpoint `POST /v1/otentikasi/perbarui-token` (Refresh Token).
- [ ] Buat Endpoint `POST /v1/otentikasi/lupa-kata-sandi` dan `reset-kata-sandi`.
- [ ] Buat Endpoint `PUT /v1/otentikasi/ganti-kata-sandi` (Dengan verifikasi password lama).
- [ ] Buat Endpoint `PUT /v1/otentikasi/token-fcm` (Menyimpan token device ke tabel pengguna).
- [ ] Uji coba semua endpoint baru menggunakan Postman/ThunderClient.

**Referensi:**
Baca `planningall.md` Bab 8.2 (Daftar Endpoint Otentikasi).
```
